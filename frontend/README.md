Como Rodar o Frontend (Vite)
Se for a primeira vez rodando o projeto no seu computador, você precisa instalar as dependências:

cd frontend
npm install

Depois que a instalação terminar, para subir o servidor local e abrir o projeto no seu navegador, execute:

npm run dev

O terminal vai liberar um link local (normalmente http://localhost:5173/). Basta clicar nele para testar a interface integrada com a sua API.

Para encerrar o frontend:
No terminal onde o Vite está rodando, pressione Ctrl + C.
